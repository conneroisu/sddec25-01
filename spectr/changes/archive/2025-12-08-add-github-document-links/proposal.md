# Change: Add GitHub Document Links to Website

## Why
Users need easy access to view project documents directly on GitHub, including both PDF and Markdown versions. This improves accessibility and allows users to see the source files in the repository.

## What Changes
- Add GitHub icon to header navigation linking to main repository
- Add GitHub icon to footer section linking to main repository
- Add GitHub PDF and Markdown icons to each document link in the documentation sections
- Add CSS styles for the new icon links

## Impact
- Affected specs: website-navigation (new capability)
- Affected code: www/index.html, www/css/site.css
