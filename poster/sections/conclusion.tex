\textbf{Human-Centered Design}

\begin{itemize}
  \item \textbf{Safety:} Fail-safe defaults, graceful degradation
  \item \textbf{Privacy:} All data processed on-device
  \item \textbf{User Control:} System supports but doesn't replace autonomy
  \item \textbf{Future Testing:} User studies with wheelchair users planned
\end{itemize}
